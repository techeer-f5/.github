## **🔧 Tech stack**

### Back-end

- Java 17
- Gradle 7.4.1
- Spring Boot 2.6.6
- Spring Security + OAuth 2.0
- Spring Data JPA
- Querydsl 5.0.0
- MySQL 8
- Lombok
- Jackson
- JUnit 5, Mockito, Hamcrest
- Spring REST Docs

### Front-end

- TypeScript
- Next.js
- MUI
- Tailwind CSS
- Zustand
- Kakao Map API
